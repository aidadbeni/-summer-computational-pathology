# Model Notes

This folder contains my notes and understanding of the computational pathology models assigned for the summer project.

The purpose of these notes is to document:

- the objective of each model
- the input and output
- the model architecture (high-level)
- the biological interpretation
- how the model relates to the overall computational pathology pipeline
- questions that arise while studying or reproducing the model

The models currently being studied are:

- HoVer-Net
- UNI
- Prov-GigaPath
